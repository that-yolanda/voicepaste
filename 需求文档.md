# **实时语音输入悬浮工具（Electron版）PRD v1.0**

---

# **一、项目目标**

## **目标**

开发一个本地桌面工具，实现：

> **按右 Command → 开始语音输入 → 实时悬浮显示 → 再按 Command → 自动粘贴到当前输入框**

---

## **一句话定义**

这是一个：

> **基于全局热键的语音输入增强工具（非输入法）**

---

# **二、核心交互设计**

## **2.1 热键定义**

| 按键               | 行为             |
| ---------------- | -------------- |
| `Command（右）` | 开始录音 / 结束录音并粘贴 |
| `Esc`            | 取消当前输入         |

---

## **2.2 状态机**

```text
idle
  ↓ Cmd
recording
  ↓ Cmd
finishing → 自动粘贴 → idle

recording
  ↓ Esc
cancelled → idle
```

---

## **2.3 用户流程**

### **流程1：正常输入**

```text
按 Cmd
→ 开始录音
→ 悬浮窗出现
→ 实时显示语音文本
→ 再按 Cmd
→ 停止录音
→ 自动粘贴
→ 悬浮窗关闭
```

---

### **流程2：取消**

```text
按 Cmd → 开始录音
→ 按 Esc
→ 丢弃结果
→ 悬浮窗关闭
```

---

# **三、功能需求**

---

## **3.1 全局热键监听**

### **要求**

* 支持监听：
  * 右 Command

* 单击触发（不是长按）

### **行为**

```text
如果当前状态 == idle → 进入 recording
如果当前状态 == recording → 进入 finishing
```

---

## **3.2 悬浮窗（核心模块）**

### **窗口特性**

* 始终置顶（always on top）
* 不抢焦点（focusable: false）
* 无边框（frameless）
* 半透明背景
* 屏幕底部居中

---

### **显示内容**

```text
[🎤 图标]  实时识别文本
```

---

### **文本显示规则**

```text
finalText（深色）
partialText（浅色）
```

合并展示：

```text
display = finalText + partialText
```

---

## **3.3 实时语音识别**

### **输入**

* 麦克风 PCM 音频流
* 16kHz / 16bit

---

### **接口**

* WebSocket 连接豆包流式语音识别

---

### **返回数据**

```json
{
  "type": "partial" | "final",
  "text": "识别内容"
}
```

---

### **处理逻辑**

```ts
onPartial(text):
  partialText = text

onFinal(text):
  finalText += text
  partialText = ""
```

---

## **3.4 自动粘贴**

### **触发**

第二次按 `Command`

---

### **流程**

```text
1. 停止录音
2. 等待最终识别结果
3. 拼接 finalText
4. 写入剪贴板
5. 模拟 Cmd + V
```

---

### **实现要求**

* 使用系统剪贴板
* 模拟键盘输入
* 不改变当前焦点窗口

---

## **3.5 取消机制**

### **触发**

按 `Esc`

### **行为**

```text
- 停止录音
- 清空所有文本
- 不执行粘贴
- 关闭悬浮窗
```

---

# **四、非功能要求**

---

## **4.1 极简原则**

* 不需要 UI 配置页
* 不需要持久化数据
* 不需要账号系统

---

## **4.2 性能要求**

* UI 刷新间隔：100~200ms
* 不允许明显卡顿
* 悬浮窗不闪烁

---

## **4.3 稳定性**

* 多次连续触发不崩溃
* WebSocket断开可恢复（简单重连即可）

# **七、关键模块说明**

---

## **7.1 热键模块**

### **逻辑**

```js
onCmdPress():
  if state == idle:
    startRecording()
  else if state == recording:
    stopRecordingAndPaste()
```

---

## **7.2 音频模块**

### **采集方式**

```js
navigator.mediaDevices.getUserMedia({ audio: true })
```

### **分片**

* 每 100~300ms 发送一段

---

## **7.3 ASR模块**

### **连接**

```text
WebSocket → 豆包流式识别
```

### **发送**

```text
音频二进制流
```

### **接收**

```json
partial / final
```

---

## **7.4 文本管理**

```ts
let finalText = ""
let partialText = ""
```

---

## **7.5 粘贴模块**

### **实现**

```js
clipboard.writeText(finalText)
simulateKeyPress("Cmd+V")
```
